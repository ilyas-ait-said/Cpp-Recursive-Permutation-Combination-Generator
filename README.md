# C++ Permutation & Combination Generator

A simple C++ console application that generates:

- Permutations of user-defined numbers
- Digit combinations (0–9) of a chosen length

The program uses recursive backtracking and saves results into text files.

----------------------------------------

## **FEATURES**

- **Recursive permutation generation**
- **Digit combination generation (0–9)**
- **Input validation handling**
- **File output support (.txt files)**
- **Interactive console menu**

----------------------------------------

## **OUTPUT FILES**

- Permutations.txt
- Digits.txt

----------------------------------------


## **COMPILATION**

g++ main.cpp -o generator
./generator

----------------------------------------

Small project built to practice combinatorics, recursion,
and file handling in C++.
